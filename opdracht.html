1.1.1 Web toepassing (CV)

De web toepassing wordt in 2 fasen uitgewerkt. Vooreerst wordt een CV uitgewerkt waarbijde focus ligt op de desktopversie. In de 2e fase wordt het CV uitgebreid zodat de website responsive wordt. De website moetuitgebreid worden om er ook op smartphones goed uit te zien.
